# UI Quality Skill

Use this skill when creating or reviewing user-facing interfaces.

## Review Matrix

Check:

- Mobile
- Tablet
- Desktop
- Light theme
- Dark theme
- Additional themes
- Loading
- Empty state
- Error state
- Keyboard use
- Focus states
- Reduced motion
- Long content
- Slow network
- No horizontal overflow
- No clipped menus
- Readable contrast

## Rules

- Use semantic HTML.
- Keep motion purposeful.
- Use theme tokens.
- Avoid hard-coded colors.
- Avoid hover-only essential actions.
- Do not create non-functional UI.
