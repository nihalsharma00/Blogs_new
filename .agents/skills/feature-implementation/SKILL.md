# Feature Implementation Skill

Use this skill when implementing a complete product feature.

## Process

1. Read project instructions and memory.
2. Define acceptance criteria.
3. Inspect existing code.
4. Identify database, API, frontend, security, and testing impact.
5. Implement a complete vertical slice.
6. Add validation and authorization.
7. Add all UI states.
8. Test.
9. Run lint, typecheck, tests, and build.
10. Report exact changes.
11. Update memory only for durable changes.

## Completion Standard

A feature is not complete when only its UI or API exists. All required layers must work together.
